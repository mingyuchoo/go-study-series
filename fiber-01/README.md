# fiber-01

## How to create a project

```bash
$ mkdir <project-name>
$ cd <project-name>
$ go mod init <project-name>
```

## How to install `fiber`

```bash
$ cd <project-name>
$ go get -u github.com/gofiber/fiber/v2
```

## How to run

```bash
$ go run server.go
```

## How to check the result

- <http://localhost:3000>
